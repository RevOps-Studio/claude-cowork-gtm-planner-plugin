# Benchmark Rules

Numbers make a plan credible — invented numbers make it fiction.

## Rules

1. **Every external number cites a source.** Benchmark reports, published
   studies, platform data, tool data (Ahrefs, Similarweb). Include source
   name and year inline.
2. **Sector first, generic second.** A B2B-services CPL benchmark beats a
   generic B2B one. Say which level was found.
3. **If no credible benchmark exists, mark `needs-benchmark`** and give a
   deliberately wide range labeled as assumption if the design cannot
   proceed without one. Never present an assumption as a benchmark.
4. **Real data beats benchmarks.** If HubSpot is connected, the company's own
   funnel numbers take precedence — benchmarks then serve as sanity checks.
5. **State the assumption chain.** Any projection (e.g. leads → meetings →
   deals) shows its conversion assumptions and where each came from.

## Presentation

Ranges, not points ("€40–90 CPL", not "€63 CPL"). Round numbers. A short
"assumptions" note under any table that uses them.
